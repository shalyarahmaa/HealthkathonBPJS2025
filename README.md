# Healthkathon BPJS 2025 - Project RISKES

Project ini dikembangkan untuk kompetisi **Innovation System Healthkathon BPJS 2025**.

## 🚀 Deskripsi Project
Project ini merupakan sistem visualisasi data kesehatan (RISKES) yang bertujuan untuk mempermudah analisis data kesehatan masyarakat secara interaktif.

## 🛠️ Fitur Utama
* Visualisasi data berbasis web (HTML/CSS/JS).
* Integrasi data dinamis melalui file JSON (`data_riskes.json`).
* Antarmuka yang responsif dan user-friendly.

## 📂 Struktur File
* `riskes.html`: Halaman utama aplikasi.
* `style.css`: Styling dan layouting.
* `script.js`: Logika aplikasi dan pengolahan data.
* `data_riskes.json`: Sumber data kesehatan.
